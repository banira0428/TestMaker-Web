# 「暗記メーカー」ブラウザ版

## 概要

スマートフォン用アプリ「暗記メーカー」の問題編集機能に特化したWebサービスです。

## 使用技術

JavaScriptフレームワーク： [Vue.js](https://jp.vuejs.org/index.html)

CSSフレームワーク： [BootStrapVue](https://bootstrap-vue.js.org/)
